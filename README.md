### 스터디 정보
* 참여자 : 이재하, 임동권, 명혁, 김진태, 김지훈, 오지원, 이호승, 허민경 (총 8명, 발표순서도 동일) 
* 교재명 : 알고리즘 문제해결 전략 (총 2권), 구종만 저
  * 1권 : 1~8장, 10장 (나머지는 2권을 먼저하고 진행)

### 목표
* 완벽히 알기는 어렵더라도 알고리즘이 이런거다라고 감 잡는것 !
